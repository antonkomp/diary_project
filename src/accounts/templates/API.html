Authorization

Get data

Request format

/api/user/get/

Response

1. 'id' - идентификатор пользователя,

2. 'last_login - дата и время последней авторизации пользователя,

3. 'username' - пользовательское имя,

4. 'first_name' - имя пользователя,

5. 'last_name' - фамилия пользователя,

6. 'is_active - доступ пользователя,

7. 'date_joined' - дата и время регистрации пользователя.

Edit

Request format

/api/user/edit/

Response

1. 'username' - пользовательское имя,

2. 'first_name' - имя пользователя,

3. 'last_name' - фамилия пользователя,

4. 'email' - почтовый ящик пользователя.


Working with user profile

Get data

Request format

/api/profile/get/

Response

1. 'user' - номер пользователя в системе,

2. 'account_type' - тип аккаунта пользователя,

3. 'birthday' - день рождения,

4. 'location' - местоположение,

5. 'website' - вебсайт пользователя,

6. 'bio' - краткая биография пользователя.

Edit

Request format

/api/profile/edit/

Response

1. 'birthday' - день рождения,

2. 'location' - местоположение,

3. 'website' - вебсайт пользователя,

4. 'bio' - краткая биография пользователя.


Working with account

Get data

Request format

/api/account/

Response

1. 'id' - идентификатор пользователя,

2. 'account_type' - тип аккаунта пользователя,

3. 'key' - ключ для изменения типа аккаунта.


Working with diary records

Get all records

Request format

/api/records/

Response

1. 'id' - идентификатор записи,

2. 'date' - дата и время создания записи,

3. 'header' - заголовок записи,

4. 'text' - текст записи,

5. 'image' - прикрепленное изображение к записи.

Create record

Request format

/api/record/create/

Response

1. 'header' - заголовок записи,

2. 'text' - текст записи,

3. 'image' - прикрепленное изображение к записи.

Edit record

Request format

/api/record/update/[pk]/

Parameter [pk] - number of the record you want to update.

Response

1. 'header' - заголовок записи,

2. 'text' - текст записи,

3. 'image' - прикрепленное изображение к записи,

4. 'delete_image' - возможность удалить изображение.

Delete record

Request format

/api/record/delete/[pk]/

Parameter [pk] - number of the record you want to delete.

Response

1. 'id' - идентификатор записи,

2. 'date' - дата и время создания записи,

3. 'header' - заголовок записи,

4. 'text' - текст записи,

5. 'image' - прикрепленное изображение к записи.
